# Herramientas

Lista completa de servicios y librerías que usamos, qué hace cada una, y por qué la elegimos.

Cada página sigue el mismo formato:

1. **¿Qué es?** — Explicación para alguien que no la conoce.
2. **¿Para qué la usamos?** — En el contexto de este proyecto.
3. **Alternativas que valoramos** — y por qué se descartaron.
4. **Enlaces** oficiales.

## Por capas

### Lenguaje y framework

- [Next.js](/herramientas/nextjs) — framework full-stack
- [TypeScript](/herramientas/typescript) — JavaScript con tipos
- [Tailwind CSS](/herramientas/tailwind) — CSS por clases utility
- [shadcn/ui](/herramientas/shadcn) — componentes copy-paste
- [Zod](/herramientas/zod) — validación de schemas

### Base de datos

- [Prisma](/herramientas/prisma) — ORM type-safe
- [PostgreSQL](/herramientas/postgresql) — la base de datos en sí
- [Neon](/herramientas/neon) — PostgreSQL serverless gestionado

### Auth y email

- [Auth.js](/herramientas/authjs) — autenticación
- [Resend](/herramientas/resend) — emails transaccionales

### Pagos y contenido

- [Stripe](/herramientas/stripe) — pagos
- [Mux](/herramientas/mux) — vídeo con DRM
- [Cloudflare R2](/herramientas/cloudflare-r2) — almacenamiento de archivos

### Despliegue y observabilidad

- [Vercel](/herramientas/vercel) — hosting + CI/CD
- [Sentry](/herramientas/sentry) — monitor de errores

### Toolchain (entorno local)

- [Volta](/herramientas/volta) — gestor de versiones de Node
- [pnpm](/herramientas/pnpm) — gestor de paquetes

### Otros

- [Bitwarden](/herramientas/bitwarden) — gestor de contraseñas
